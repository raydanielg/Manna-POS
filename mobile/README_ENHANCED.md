MannaPOS Mobile
=========

A modern, powerful point-of-sale mobile application built with Flutter. This app brings Odoo-like elegance and enterprise-level features to your mobile devices, enabling businesses to manage sales, inventory, and customer relationships from anywhere.

## Overview

MannaPOS Mobile provides a comprehensive mobile solution for managing point-of-sale operations with a focus on:

- **Enterprise-grade POS functionality** - Advanced sales processing with real-time inventory management
- **Intuitive user experience** - Clean, modern UI with smooth animations and intuitive navigation
- **Powerful business insights** - Dashboard analytics, reporting, and business intelligence
- **Cross-platform compatibility** - iOS, Android, and web support

## Key Features

### 🎯 Core POS Experience
- **Intuitive Product Catalog** - Smart search, barcode scanning, and categorized product browsing
- **Professional Cart Management** - Real-time quantity adjustments, discounts, and tax calculations
- **Flexible Payment Options** - Cash, card, mobile money, and credit payments with change calculation
- **Instant Receipt Generation** - Professional, shareable receipts with business branding
- **Offline Capability** - Basic operations work offline with sync when back online

### 📊 Business Intelligence
- **Live Dashboard** - Real-time sales metrics, inventory alerts, and business KPIs
- **Interactive Sales Analytics** - 7-day trend charts with hover effects and smooth animations
- **Top Products Report** - Revenue-generating products with detailed analytics
- **Sales Performance Tracking** - Daily, weekly, and monthly sales reporting
- **Payment Method Analytics** - Breakdown of payment methods with visual charts

### 🏪 Inventory Management
- **Product Management** - Complete CRUD operations for products, variants, and categories
- **Stock Control** - Real-time inventory levels with low stock alerts
- **Price Management** - Dynamic pricing, discounts, and tax configurations
- **Supplier Management** - Track suppliers and purchase history

### 👥 Customer Relations
- **Customer Database** - Complete customer profiles with purchase history
- **Loyalty Programs** - Points-based rewards and customer segmentation
- **Communication Tools** - SMS integration and email templates

### 📈 Advanced Features
- **Multi-business Support** - Manage multiple businesses from a single app
- **Role-based Access** - Secure permissions for different user roles
- **Offline Mode** - Full functionality with automatic sync
- **Push Notifications** - Real-time alerts for important events
- **Barcode/QR Scanner** - Lightning-fast product lookup

## Getting Started

```bash
# Clone the repository
flutter clone https://github.com/your-org/mannapos-mobile

# Install dependencies
flutter pub get

# Run on Android
flutter run

# Run on iOS
flutter run --platform ios

# Run on web
flutter run --web-renderer html
```
